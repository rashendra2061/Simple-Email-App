# GMAIL "CLONE" USING SQL DATABASE

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DB_HOST`
`DB_USERNAME`
`DB_PASSWORD`
`DB_NAME`

## Tech Stack

**Client:** React , Svelte
**Server:** Node , Express , MySQL
